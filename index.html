BigDoc

BigDoc was created with the hopes of combating the increasing costs of preventitive healthcare. BigDoc consists of an extremely low cost wearable device which gathers medical data, a powerful machine learning algorithm to process this data, and an encrypted messaging system to send the results to healthcare services.

I created the machine learning algorithm using python. The algorithm scans the database of medical data and compares a user's data to that of an average healthy person's. The algorithm is able predict if there is likely to be any medical issue in the future based on the user's past history.
